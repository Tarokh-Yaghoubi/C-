# FFT Project , Qt , C++ programming

```

- Added Write , Read . Generate and Close click buttons .
- As of now , This program generates 10 * 1024 Float numbers randomly and stores them in a text box 
- By clicking the WRITE button it will write those random numbers into a file 
- By clicking the READ button it will show you the content of that file 
- If you click on CLOSE button , it will exit the program .
```
